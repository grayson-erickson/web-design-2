About Me

My name is Grayson Erickson, and this is a website I created to document a TTRPG world I've been building!

This site is far from finished, but what I have now should be enough for my project. I've decided that I'll continue to expand this even after the class ends, so if you think this is a neat idea for a world, you can always check back in and see if I've made any changes! I have had this world stewing in my brain for a few years, after all.
